# ShopBridge × UCP Demo

A single-page demo project that showcases a polished, portfolio-style experience for the **Universal Commerce Protocol (UCP)**.

## Overview

This repository provides a visual demo (`index.html`) designed to communicate UCP concepts and capabilities in an accessible format.

UCP is an open standard for interoperable commerce integrations across platforms, businesses, and providers. Learn more at [ucp.dev](https://ucp.dev).

## What this demo highlights

- UCP-aligned narrative for agentic commerce use cases
- Capability-oriented presentation (for example: checkout, identity, order workflows)
- A modern UI for presenting protocol concepts and implementation maturity

## Project structure

- `index.html` — standalone demo page (HTML/CSS/JS in one file)

## Getting started

Because this is a static demo, no build step is required.

1. Clone the repository.
2. Open `index.html` directly in your browser.

Optional local server:

```bash
python3 -m http.server 8000
```

Then open `http://localhost:8000`.
